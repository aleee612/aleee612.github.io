### 接口类型
| 类型   | 读写单位     | 特点                                  | 常见设备<br>           |
| ---- | -------- | ----------------------------------- | ------------------ |
| 块设备  | 固定大小的数据块 | 支持随机访问<br>数据传输速率较高                  | HDD，SSD，USB闪存      |
| 字符设备 | 连续的字符流   | 不支持随机访问<br>不依赖数据块的概念                | 键盘，鼠标，<br>串口终端，打印机 |
| 网络设备 | 数据包      | 注重 数据的高速传输 和 实时性<br>涉及中断机制和DMA来提高效率 | 网络适配器，调制解调器        |

### 设备管理
#### 缓冲区
作用：临时存放数据，使 IO操作 与 其他处理操作 可以相对独立地进行
位置：内存
##### 磁盘高速缓存
> 即在RAM与磁盘间的“cache”
##### 缓冲区实现：多缓冲与单缓冲
| 对比项          | 单缓冲（Single Buffer） | 多缓冲（Multi Buffer，常见双缓冲） |
| ------------ | ------------------ | ----------------------- |
| 缓冲区数量        | 1 个                | ≥2 个                    |
| 并行能力         | 不能并行               | 可以并行                    |
| CPU 与 I/O 关系 | 串行执行               | 可重叠执行                   |
| 数据流过程        | 读 → 处理 → 再读        | 一边读一边处理                 |
| CPU 等待情况     | 容易等待               | 等待时间减少                  |
| 吞吐量          | 较低                 | 较高                      |
| 复杂度          | 简单                 | 较复杂                     |
| 典型应用         | 简单设备               | 磁盘、网络、音视频               |
#### SPOOLing 技术
- 即将数据暂时存放在磁盘等中间存储区域
- 由专门的后台程序将其送往目标外设
- 从而达到用户提交任务后立刻可以继续其他操作（即cpu与外设并行）的结果
