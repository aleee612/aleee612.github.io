##### 数据通路
> 执行指令的过程中内部数据流动的路径

| 分类     | 特点                                                           | 举例                          |
| ------ | ------------------------------------------------------------ | --------------------------- |
| 组合逻辑元件 | 输出仅取决于当前的输入，而不受存储器或时钟信号的影响                                   | 算数逻辑单元（ALU）、译码器、多路选择器、三态门   |
| 时序逻辑元件 | 输出不仅与该时刻的输入有关，还与该时刻以前的输入有关，并且 **具备状态存储** 功能，必须在 **时钟节拍** 下工作 | 通用寄存器组、程序计数器、状态/移位/暂存/锁存寄存器 |
### 寄存器
| 类型    | 寄存器         | 作用            |
| ----- | ----------- | ------------- |
| 通用寄存器 | AX、BX、CX、DX | 存储操作数或运算结果    |
| 段寄存器  | CS、DS、SS、ES | 保存各段的起始地址     |
| 指针寄存器 | SP、BP、IP    | 指向栈或指令位置      |
| 变址寄存器 | SI、DI       | 用于数组、字符串等数据访问 |
| 标志寄存器 | FLAGS       | 存储运算状态和控制标志   
#### 段寄存器
> 用于指向程序运行时不同内存段的起始位置。

| 寄存器 | 含义            | 对应段 |
| --- | ------------- | --- |
| CS  | Code Segment  | 代码段 |
| DS  | Data Segment  | 数据段 |
| SS  | Stack Segment | 栈段  |
| ES  | Extra Segment | 附加段 |
- 程序逻辑上通常包含四个段：
	代码段：存放指令  
	数据段：存放数据  
	栈段：函数调用和局部变量  
	附加段：额外数据空间
#### 指针寄存器
| 寄存器 | 含义                  | 作用        |
| --- | ------------------- | --------- |
| SP  | Stack Pointer       | 指向栈顶      |
| BP  | Base Pointer        | 指向当前函数栈底  |
| IP  | Instruction Pointer | 指向下一条指令地址 |
#### 变址寄存器
| 寄存器 | 含义                | 作用      |
| --- | ----------------- | ------- |
| SI  | Source Index      | 源变址寄存器  |
| DI  | Destination Index | 目的变址寄存器 |
#### 标志寄存器（FLAGS）
> 标志寄存器用于保存 CPU 运算产生的状态信息。
> 8086 的 FLAGS 是 **16 位寄存器**，其中 **9 个标志位有效**。
#### 状态标志
| 标志  | 含义   |
| --- | ---- |
| CF  | 进位标志 |
| ZF  | 零标志  |
| SF  | 符号标志 |
| OF  | 溢出标志 |
| PF  | 奇偶标志 |
| AF  | 辅助进位 |
> 溢出判断规则：
	两个操作数符号相同  
	结果符号不同 → 溢出
#### 控制标志
| 标志  | 含义      |
| --- | ------- |
| IF  | 中断允许标志  |
| TF  | 单步调试标志  |
| DF  | 字符串方向标志 |
说明：
- **[[IF中断允许位|IF]] = 1** ：允许中断
- **TF = 1** ：单步执行模式
- **DF = 0** ：字符串从低地址向高地址
#### 特殊寄存器
| 寄存器 | 作用                |
| --- | ----------------- |
| IR  | 指令寄存器，存放当前指令      |
| T   | 暂存器，临时保存 ALU 的操作数 |
| MAR | 存储访问的内存地址         |
| MDR | 存储从内存读出的数据        |
#### 暂存器 T
因为 **内部总线一次只能传一个数据**，所以 ALU 运算过程通常是：
	操作数1 → T  
	操作数2 → ALU  
	ALU(T , 操作数2)
#### 可见寄存器 vs 不可见寄存器
**汇编程序员可见寄存器**：
	AX BX CX DX  
	CS DS SS ES  
	SI DI BP SP  
	IP  
	FLAGS
**不可见寄存器**：
	MAR  
	MDR  
	IR  
	T
### CPU 引脚（Pin）
> CPU 引脚是处理器与外部设备通信的接口。
#### 地址 / 数据复用
| 引脚       | 方向  | 作用         |
| -------- | --- | ---------- |
| AD0–AD15 | 双向  | 地址 / 数据复用线 |
| A16–A19  | 输出  | 高位地址线      |
8086：
数据总线 = 16 位  
地址总线 = 20 位  
最大寻址空间 = 1MB
#### 中断相关引脚
| 引脚    | 作用     |
| ----- | ------ |
| NMI   | 不可屏蔽中断 |
| INTR  | 可屏蔽中断  |
| INTA  | 中断响应   |
| RESET | CPU复位  |
#### 控制信号
| 引脚    | 作用     |
| ----- | ------ |
| RD    | 读信号    |
| WR    | 写信号    |
| ALE   | 地址锁存   |
| READY | 设备准备信号 |
| HOLD  | 总线请求   |
| HLDA  | 总线应答   |
