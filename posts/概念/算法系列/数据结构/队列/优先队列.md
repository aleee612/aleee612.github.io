### cpp STL实现
```cpp
#include <iostream>
#include <queue>
using namespace std;

int main() {
	priority_queue<int> pq; // 默认大根堆
	priority_queue<int, vector<int>, less<int>> pq; // 大根堆
	priority_queue<int, vector<int>, greater<int>> pq; // 小根堆

	pq.push(3);      // 插入
	pq.pop();        // 删除堆顶
	pq.top();        // 访问堆顶
	pq.empty();      // 判空
	pq.size();       // 元素个数
	cout << pq.top() << endl; // 10
}
```
#### 时间复杂度
| 操作   | 复杂度      |
| ---- | -------- |
| push | O(log n) |
| pop  | O(log n) |
| top  | O(1)     |
