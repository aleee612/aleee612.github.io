核心思路：
- 通过前缀和，实现在$O(n)$内多轮的对元素的操作
- 本质：积分与微分
> 可以认为差分是前缀和的逆运算
### 练习
##### 一维差分：1109. 航班预订统计
![[Pasted image 20260312145307.png|489]]
```cpp title="1109" fold
class Solution {
public:
    vector<int> corpFlightBookings(vector<vector<int>>& bookings, int n) {
        vector<int> ans(n+1,0);
        for(int i = 0;i<bookings.size();i++){
            ans[bookings[i][0] - 1] += bookings[i][2];
            ans[bookings[i][1]] -= bookings[i][2];
        }
        for(int i = 1;i<n;i++){
            ans[i] += ans[i-1];
        }
        vector<int> anss(n);
        for(int i = 0;i<n;i++){
            anss[i] = ans[i];
        }
        return anss;
    }
};
```
- 注意点：由于有+1操作，需要防止数组访问溢出
