> 大范围的问题 拆分成 若干小问题 来求解
#### 基于两侧端点
**问题描述：**
- 求一个序列`s`中增加多少字符可以获得回文串
**核心思想：**
1. 令`i`，`j`分别为序列`s` **两端** 的下标
2. 对于`s[i]==s[j]`，则认为长度+2
3. 若不等，则分两种情况取最小值
	1. 选择左侧，在`s`右侧添加字符凑回文`dp[i+1][j]`
	2. 选择右侧，在`s`左侧添加字符凑回文`dp[i][j+1]`
```cpp title=转化公式
if (s[i] == s[j])
    dp[i][j] = dp[i+1][j-1] + 2;
else
    dp[i][j] = min(dp[i+1][j], dp[i][j-1]) + 1;
```
#### 基于范围划分点
**问题描述：**
- 对序列 s 的任意区间 `[i..j]`，求最长回文子序列长度（或最少插入字符得到回文）
- 核心思想是**先处理小区间，再合并成大区间**
**核心思想：**
1. 令 `len = j - i + 1` 为区间长度，从小到大枚举
2. 对每个左端点 `i`，右端点 `j = i + len - 1`
3. 转移依赖的是比当前区间 **短** 的子区间：
    - 如果 `s[i] == s[j]` → `dp[i][j] = dp[i+1][j-1] + 2`
    - 反之 → `dp[i][j] = max(dp[i+1][j], dp[i][j-1])`
4. 小区间先算（长度 1 或 2），保证大区间依赖的子区间已经有值
