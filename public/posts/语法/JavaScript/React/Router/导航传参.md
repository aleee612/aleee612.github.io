**传参的本质，就是让页面根据不同的数据动态渲染内容，而不是一张写死的死板页面。**
——不然的话，如果每篇文章、每个商品、每次搜索都写一个新的页面，项目会爆炸💥，又难维护又浪费时间。
所以，我们通常会**用一个模板页面**，然后根据不同的参数去**请求不同的数据**，再动态显示出来。
```
用户点击链接（带参数）
        ↓
navigate('/article/1234')
        ↓
路由变化，跳到 ArticleDetail 页面
        ↓
ArticleDetail 页面用 useParams() 或 useLocation() 取到参数
        ↓
根据参数（比如文章id）发请求拉数据
        ↓
拿到数据后，动态渲染页面内容
```
### SerchParams传参
- `navigate`中参数的形式为`?参数1=数据&参数2=数据`
#### 在传参的界面中
```js
import { useNavigate } from "react-router-dom";
function Login(){
  const navigate = useNavigate();
  return(
    <div>
      <h1>Login</h1>
      <button onClick={()=>{navigate("/article?id=1001&name=jack")}}>searchParams传参</button>
    </div>
  )
}
export default Login;
```
#### 在接收参数的页面中
```js
import { useSearchParams } from "react-router-dom";
function Article(){
    const [params] = useSearchParams();//定义一个param接收一个对象
    const id = params.get("id");//使用get获取id
    const name = params.get("name");//name同理
    return(
        <div>
            <h1>Article - {id} - {name}</h1>
        </div>
    )
}
export default Article;
```
### params传参
#### 传参的界面中
```js
import { useNavigate } from "react-router-dom";
function Login(){
    const navigate = useNavigate();
    return(
        <div>
            <h1>Login</h1>
            <button onClick={()=>{navigate("/article/1001/jack")}}>Params传参</button>
        </div>
    )
}
export default Login;
```
#### 在接收参数的页面中
```js
import { useNavigate, useSearchParams,useParams } from "react-router-dom";
function Article(){
    const params = useParams();
    const {id,name} = params;
    return(
        <div>
            <h1>Article - {id} - {name}</h1>
        </div>
    )
}
export default Article;
```
#### 在router中
```js
//导入项目的根组件
import Login from '../page/login'
import Article from '../page/article'
import { createBrowserRouter } from 'react-router-dom';

const router = createBrowserRouter([
    {
        path: '/login',
        element: <Login />,
    },
    {
        path: '/article/:id/:name',//在这里改传进来的路径
        element: <Article />,
    }
])

export default router
```
