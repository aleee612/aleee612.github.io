### 常见指令
| 指令类别   | 指令        | 含义     | 说明                   |
| ------ | --------- | ------ | -------------------- |
| 数据传输指令 | MOV       | 数据传送   | 在寄存器、内存或立即数之间复制数据    |
| 栈操作指令  | PUSH      | 入栈     | 将数据压入栈顶              |
|        | POP       | 出栈     | 从栈顶弹出数据              |
| 算术运算指令 | ADD       | 加法     | 第一个操作数 + 第二个操作数      |
|        | SUB       | 减法     | 第一个操作数 − 第二个操作数      |
|        | MUL       | 无符号乘法  | 乘积存放在 `EDX:EAX`      |
|        | IMUL      | 有符号乘法  | 支持多种格式               |
|        | DIV       | 无符号除法  | 商→EAX，余数→EDX         |
|        | IDIV      | 有符号除法  | 商→EAX，余数→EDX         |
| 位运算指令  | AND       | 按位与    | 指定位保留，其余清零           |
|        | OR        | 按位或    | 将某些位置为1              |
|        | XOR       | 按位异或   | 位翻转                  |
|        | NOT       | 按位取反   | 所有位取反                |
| 自增自减   | INC       | 自增     | 操作数 +1               |
|        | DEC       | 自减     | 操作数 −1               |
| 比较指令   | CMP       | 比较     | 本质是减法但不保存结果，只更新标志位   |
| 移位指令   | SHL / SAL | 左移     | 相当于乘2                |
|        | SHR       | 逻辑右移   | 高位补0                 |
|        | SAR       | 算术右移   | 保留符号位                |
|        | ROL       | 循环左移   | 高位移到低位               |
|        | ROR       | 循环右移   | 低位移到高位               |
| 控制转移   | JMP       | 无条件跳转  | 跳转到指定地址              |
| 条件跳转   | JE / JZ   | 相等跳转   | `ZF=1`               |
|        | JNE / JNZ | 不等跳转   | `ZF=0`               |
|        | JG        | 大于跳转   | `ZF=0 且 SF=OF`       |
|        | JL        | 小于跳转   | `SF≠OF`              |
|        | JGE       | 大于等于   | `SF=OF`              |
|        | JLE       | 小于等于   | `ZF=1 或 SF≠OF`       |
| 子程序调用  | CALL      | 调用子程序  | 返回地址压栈               |
|        | RET       | 返回     | 从栈中取返回地址             |
| I/O 指令 | IN        | 输入     | 从端口读数据               |
|        | OUT       | 输出     | 向端口写数据               |
| 字符串操作  | MOVS      | 字符串复制  | `ES:EDI ← DS:ESI`    |
|        | LODS      | 加载字符串  | `AL/AX/EAX ← DS:ESI` |
|        | STOS      | 存储字符串  | `ES:EDI ← AL/AX/EAX` |
|        | CMPS      | 字符串比较  | 比较两个字符串              |
| *陷阱指令* | *INT*     | *软件中断* | *触发系统调用或异常*          |
| 中断控制   | CLI       | 关闭中断   | 清除 IF 标志             |
|        | STI       | 开启中断   | 设置 IF 标志             |
