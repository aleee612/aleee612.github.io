### cpp STL实现双端队列
```cpp
#include <iostream>
#include <deque>
using namespace std;

int main() {
    deque<int> dq;

    dq.push_back(1);   // 尾部插入
    dq.push_back(2);
    dq.push_front(0);  // 头部插入

    cout << dq.front() << endl; // 0
    cout << dq.back() << endl;  // 2

    dq.pop_front();  // 删除头
    dq.pop_back();   // 删除尾

    cout << dq.size() << endl;
}
```
#### 常见函数
|函数|作用|
|---|---|
|`push_back(x)`|尾部插入|
|`push_front(x)`|头部插入|
|`pop_back()`|删除尾|
|`pop_front()`|删除头|
|`front()`|访问头元素|
|`back()`|访问尾元素|
|`size()`|长度|
|`empty()`|是否为空|
|`dq[i]`|随机访问|
### 核心思路
这类题一般主要针对想要随时获取当前最大值的情况
- 在双端队列中，只维护当前队列中最大的值在队首的位置
- 对于右侧增加的值，将其与队尾值作对比，若≥队尾则将队尾值弹出，直至队尾不再小于增值
- 对于左侧减去的值，则根据题意
- 保证队列中始终具有单调的特性
### 题目练习
#### leetcode 862
#### ![[Pasted image 20260316163325.png]]
```cpp
class Solution {
public:
    int shortestSubarray(vector<int>& nums, int k) {
        int n = nums.size();
        long long pre[100001];
        pre[0] = 0;
        for(int i = 1; i <= n; i++)
            pre[i] = pre[i-1] + nums[i-1];
        int dq[100001];
        int l = 0, r = 0;
        int Min = INT_MAX;
        for(int i = 0; i <= n; i++) {
            while(r - l > 0 && pre[i] - pre[dq[l]] >= k) { // 当前前缀和与队列中的首位元素差k以上
                Min = min(Min, i - dq[l]); // 记录当前数
                l++; // 往后移一位
            }
            while(r - l > 0 && pre[i] <= pre[dq[r-1]]) { // 当前前缀和小于前缀和的最后一位
                r--; // 回退直至不再小于，而后使用r++直接覆盖当前元素
            }
            dq[r++] = i;
        }
        return Min == INT_MAX ? -1 : Min;
    }
};
```
核心思路：
1. 使用前缀和，使判断条件减少
2. 使用 `l` 表示左指针，`r` 表示右指针，`i` 来遍历数组中所有的前缀和
3. 当需要从左侧出队列时，将 `l++` 
4. 当需要从左侧出队列时，将 `r--` 
5. 当需要从右侧入队列时，将 `i` 放入 `dq[r++]` 中，这样直接达成覆盖之前 `r--` 出队列数据的效果
6. `dq` 数组中存储的依旧是下标

```cpp
class Solution {
public:
    int findMaxValueOfEquation(vector<vector<int>>& points, int k) {
        int n = points.size();
        int sub[100000];
        for(int i = 0;i < n; i++){
            sub[i] = points[i][1] - points[i][0]; // 差值数组
        }
        int dq[100000];
        int l = 0,r = 0;
        int Max = INT_MIN;
        for(int i = 0;i < n; i++){
            while(r-l > 0 && points[i][0] - points[dq[l]][0] > k){
                l++;// 左侧出队列
            }
            if(r-l > 0) {
                int x = points[i][0];
                int y = points[i][1];
                Max = max(Max,sub[dq[l]] + x + y);
            }
            while(r-l > 0 && sub[i] >= sub[dq[r-1]]){
                r--;// 右侧出队列
            }
            dq[r++] = i;
        }
        return Max;
    }
};
```
核心思想：
1. 本题与上一题整体思路相似，使用 `sub` 维护 `x-y` 差值
2. 需注意每次最大值的计算是在左侧出队列时，而非右侧后
3. 需注意 `sub` 的访问方式，由于 `dq` 保存的一直为下标，因此需使用 `sub[dq[l]]` 等来访问
