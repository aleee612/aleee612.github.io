### 01背包
**常见题目：**
- 有背包容量W，n个物品分别价值为`v[i]`，重量为`w[i]`
- 问如何装能使背包中的物品价值最大
**基本思路：**
1. 使用二维dp，第`i`行代表当前判断的为第`i`个物品，第`j`列代表当前为`j`容量，存储为当前总价值，则`dp[n][W]`为所求值
2. 由此可以分为两种条件筛选：
	1. 不存放第`i`个物品 -> 则当前dp表延续上一层的值，即`dp[i][j] = dp[i-1][j]`
	2. 存放第`i`个物品 -> 则当前dp表使用上一层中，自己能够存下的值（需做判断`j-w[i]>=0`），即`dp[i][j] = dp[i-1][j-w[i]] + v[i]`
	3. 在步骤1，2中取最大值
**空间压缩：**
3. 仅使用一维dp表，在进行筛选时从右往左逆向选择，只判断存放第`i`个物品的情况
4. 若可以存放第`i`个物品，则在原值与`dp[j-w[i]] + v[i]`中取最大值
5. 也就是通过逆向，省去了`dp[i][j] = dp[i-1][j]`的过程，并节省了空间
```cpp title=空间压缩代码 fold
for (int i = 0; i < n; i++) {
    for (int j = V; j >= w[i]; j--) {
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
    }
}
```
### 有依赖的背包
**常见题目：**
- 部分元素需要其前置绑定元素已选择的情况下，才能被选择
- `lost[]`：物品代价，`val[]`：物品价值，`q[]`：物品前置绑定的元素，若无则为0
**数据结构：**
- `king[]`：是否为主元素（无前置绑定元素）
- `fans[]`：该（主）元素有几个被绑定的元素
- `follow[][]`：该主元素下，有哪些被绑定的元素，以数组方式呈现
**基本思路：**
1. 依旧第`i`行代表当前判断的为第`i`个主商品，第`j`列代表当前为`j`容量
2. 使用`p`记录当前使用的是哪个主元素，条件筛选时仅讨论`p`，讨论完后将`p`指向下一个主元素（即`King[p] == true`）
3. 对于`p`下的被绑定元素，通过`fans`中数值记录其是否存在
4. 对被绑定元素进行分别dp
**空间压缩：**
5. 与01背包整体类似，不过是必定选问主元素
6. 不需要`p`元素专门记录
### 分组背包
**常见题目：**
- 物品被分为`k`组，每个组里只允许取一个
**核心思路：**
1. 第`i`行代表当前判断的为第`i`个组，第`j`列代表当前为`j`容量
2. 使用`arr[]`表示当前组
3. 分为要`i`组的商品和不要`i`组的商品
	1. 对于不要 -> 则直接跳过该组
	2. 对于要 -> 则对组内每一个元素做dp展开，取最佳值
**空间压缩：**
4. 类依赖背包，必定选问的为组
```cpp title=分组
for (int i = 0; i < n; i++) { // 枚举组  
	for (int j = V; j >= 0; j--) { // 倒序容量（防止重复选同组）  
		for (int k = 0; k < v[i].size(); k++) { // 枚举组内物品  
			if (j >= v[i][k]) {  
				dp[j] = max(dp[j], dp[j - v[i][k]] + w[i][k]);  
			}  
		}  
	}  
}
```
### 完全背包
**常见题目：**
- 每个物品可以重复选任意个
**核心思路：**
1. 依赖的不再是`dp[i-1][j-w[i]]`，而是`dp[i][j-w[i]]`
**空间压缩：**
2. 与01背包的代码很像，不同点在于内层的循环条件，`j`不再是从大到小，而是从小到大
3. 因为完全背包空间压缩做法中，覆盖是从左往右而不再是从右往左，这样才能使递推的数值是`i`而不是`i-1`
```cpp title=完全背包压缩 fold
for (int i = 0; i < n; i++) {
    for (int j = w[i]; j <= V; j++) {
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
    }
}
```
### 多重背包
**常见题目：**
- 每个物品可以重复选多个，但有上限`c[i]`
**核心思路：**
1. 分为要`i`和不要`i`
	1. 对于不要 -> 则直接跳过该物品
	2. 对于要 -> 枚举所有可能重复选择的个数`k`，通过`dp[i-1][j - k*w[i]] + k*v[i]`实现
2. 在“要”的分支中，取最大
**空间压缩：**
3. 同01，使用逆序
```cpp title=多重 fold
for(int i = 1; i <= n; i++){
	for(int j = t; j >= 0; j--){ // 逆序
		for(int k = 1; k <= c[i] && w[i] * k <= j; k++){
			dp[j] = max(dp[j],dp[j - k * w[i]] + k * v[i]);
		}
	}
}
```
##### 优化枚举：
4. 经过**二进制**分组，转化为01背包
5. 将同一物品中的所有 $2^x$ 个数的总价值，总体积视为一个新物品
6. 而对于不等于 $2^x$ 个数的，使用多个新物品相加的方式实现（dp表的构建是转移叠加得出的，因此可以实现`dp[j]=dp[j-2](dp[j−3]+v1)​+v2​`的效果）
7. 对于拆分出来的最后一个数，将不再是 $2^x$ ，而是总数 `k` - 所有被拆分出去的 $2^x$ ，以达到最终所有相加等于 `k` 的效果
**数据结构：**
- `v[]`数组记录二进制转化后的价值
- `w[]`记录二进制转化后的重量
```cpp title=优化
#include <bits/stdc++.h>
using namespace std;

const int N = 10010;

int dp[N];   // 一维DP（滚动数组）
int w[N];    // 拆分后的重量
int v[N];    // 拆分后的价值

int main() {
    int n, V;
    cin >> n >> V;  // n种物品，背包容量V

    int cnt = 0;  // 拆分后的物品数量

    for (int i = 1; i <= n; i++) {
        int weight, value, k;
        cin >> weight >> value >> k;

        // 二进制拆分
        for (int j = 1; j <= k; j <<= 1) {
            w[++cnt] = j * weight;
            v[cnt] = j * value;
            k -= j;
        }
        // 剩余部分（不是2^x）
        if (k > 0) {
            w[++cnt] = k * weight;
            v[cnt] = k * value;
        }
    }

    // 01背包（倒序）
    for (int i = 1; i <= cnt; i++) {
        for (int j = V; j >= w[i]; j--) {
            dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
        }
    }

    cout << dp[V] << endl;
}
```
